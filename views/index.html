Soundcloud Playlist Downloader

Using this website, you can download all of the publicly available soundcloud playlists on any user's profile. It will deliver the playlists to you in the form of a zip archive that contains one directory for each individual playlist downloaded - where inside each playlist's directory will be all of the audio files for the tracks in that playlist, with corresponding artwork and artist (uploader) information embedded.
